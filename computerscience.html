Data Structures and Algorithms

What is Sorting?

Sorting algorithms simply reorder elements(integers, numbers, strings, etc) of an array (a list) in a certain order(increasing, decreasing, lexicographical, etc). There are many different sorting algorithms, and each has its own advantages and limitations.

What is a linked list?

A linked list is a data structure consisting of a group of certices which together represent a sequence. Under the simplest form, each vertex is composed of data and areference (in other words, a link) to the next vertex in the sequence.

What is a hash table?

A hash table is a data structure to map keys to values. It uses a hash function to ma keys into a small range of indices(typically [0..hash_table_size-1].

Graphs and Trees

What is a Binary Search Tree?

A Binary Search Tree(BST) is a binary tree in which each vertex has only up to 2 children that satisfy BST property. All vertices in the left subtree of a vertext must hold a value smaller than its own and all vertices in the right subtree of a vertext must hold a value larger than its own.

What is a Binary Heap?

A Binary Heap is a complete binary tree that maintains the Max Heap property. Binary Heap is one possible data structure to model an efficient Priority Queue.

Why is a Graph Structure?

A graph is made up of 'vertices' (or 'nodes') and 'edges' (or 'lines') that connect those vertices. A gaph may be undirected (or bidirectional), meaning that there is no distinction between the two vertices associated with each edge, or its edges may directed from one vertex to another but no neccessarily in the other direction. A graph can be weighted by assigning a weight to each edge, which represent numerical values associated with that connection, or a graph can be unweighted (either all edges have unit weight 1 or all edges have the same constant weight).

What is Graph Traversal?

Given a graph, we can use DFS(Depth-First-Search) or BFS(Breath-First-Search) to traverse the graph and explore the features/properties of the graph. Each graph traversal algorithm has its own characteristics.

Recursion and Other Uses of Trees

What is Recursion?

Recursion is a method where the solution to a problem depends on solutions to smaller instances of the same problem(as opposed to iteration).

What is a Minimum Spanning Tree?

A Spanning Tree(ST) of a conected undirected weighted graph G is a subgraph of G that is a tree and connects(spans) all vertices of G. A graph can have multiple STs. A Minimum Spanning Tree(MST) of G is an ST of G that has the smallest total weight among the various STs. A graph G can have multiple MSTs but the MST weight is unique.

What is Single Source Shortest Path?

In a Single Source Shotest Path problem, we aim to find the shortest paths from a particular single source vertex to all other vertices in a directed weighted graph(if such paths exist). Different algorithms can be used depending on the nature of the directed weighted graph, i.e. weighted/unweighted, with/without negative weight/cycle.
